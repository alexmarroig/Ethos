ETHOS Architecture Overview



This repository is a monorepo.



Applications:



apps/ethos-clinic

Backend responsible for clinical operations.



apps/ethos-control-plane

Administrative backend.



apps/ethos-desktop

Electron desktop client used by psychologists.



apps/ethos-mobile

Mobile application built with Expo.



apps/ethos-transcriber

Worker responsible for Whisper transcription.



Shared packages:



packages/shared

Common utilities and types.



packages/ethos-sdk

SDK used to interact with APIs.



Frontend:



Frontend/

Lovable-based React interface.
